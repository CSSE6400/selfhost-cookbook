## Manual

```shell

$ sudo yum install git python3 nodejs
$ cd /var
$ sudo mkdir app
$ cd app
$ sudo git clone https://github.com/m0ngr31/DailyNotes
$ sudo pip install -r requirements.txt
$ sudo ./verify_env.py
$ ./run.sh
$ cd client && npm run serve

```
